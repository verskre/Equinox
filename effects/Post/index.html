    MFIMotionBlur

    Parameters

    0. Macros

    str - No definition provided
    Graphics Platform (RTX PC vs Mac) menu - No definition provided
    Motion Blur Mode menu - Critical mode selection. Read-only for macro system.
    GEO Render pulse - Uses linked Render TOPs / GEO workflow to stablize vectors for Optical Flow.
    TOPs Only pulse - Only uses TOPs generate vectors with Optical Flow.
    TOPs Only (+ Custom Vector Input) pulse - Uses TOPs and allows for manual vector input on 2nd input pin.
    WIP - POPs Vectors pulse - Will allow for POPs based workflow.
    WIP - World Space Vector System pulse - Will allow for world space based workflow.
    Coming Soon - Maximum pulse - WIP
    Coming Soon - Medium pulse - WIP
    Coming Soon - Low pulse - WIP
    Motion Blur Amount float - Macro for changing motion blur amount (shutter speed time). Can be increased well above 2 by middle mouse dragging, or manual entry of values.
    Motion Blur Samples (Quality) int - Macro for changing motion blur sample count (amount of pseudo-frames to blend). Can be increased above 128 at the cost of performance. Certain values have rounding errors, so be aware if you want to animate this value.
    Motion Vector Smoothing float - Macro for blurring overall vector map. This decreases noise greatly at the cost of some details. You can clean up the blur on Page 2. Blur > Refine Vector Edge settings.
    Reset: Macros pulse - No definition provided

    1. Setup

    Auto Discover Render / Camera toggle - No definition provided
    Render TOP top - This is the Render TOP that you are using for your Raw Render.
    Render Camera object - This is the same Camera that you are using for your Raw Render.
    Geometry Method menu - Local Geometry: GEOs next to this COMP Parent Geometry: GEOs above this COMP in Parent COMP Select Geometry Path: Specify GEOs location with path All Geometry with Tag: Uses OP Tags to find GEOs anywhere in project Render TOP Geometry: Uses Render TOP's Geometry list to find GEOs
    Geometry Path str - Manual path to Geometry.
    Geometry Tag str - Manual OP Tag for accessing Geometry anywhere in the project. Very nice workflow for selectively choosing certain Geometry.
    Vector System menu - Determines which vector system to use. Controlled by workflow macros.
    Geo Sample Density menu - Changes the noise override scale to allow for larger/smaller tiling when working with various large/small Geometry scales.
    Geo Sample Sharpness Filter menu - Changes the way that noise material override is filtered. (This is the setting that controls the blurriness that occurs when looking along an angle near parallel to a polygon. See also: https://derivative.ca/UserGuide/Texture_Filtering
    WIP - Transparency menu - WIP - Looking for solution for blending transparency with material override. Likely GLSL MAT solution if anyone is interested in assisting, please reach out.
    Overscan toggle - Enables overscan at a rate of either 0.9 or 1.1x scale. (Used primarily for Render GEO workflow to avoid vectors at edges from popping in/out.
    Overscan Extend Method menu - Changes how overscan will extend. Mirror is optimal so it doesn't affect the opposite side of the image. Can be fun to experiment with for glitches.
    Show Delay Debug toggle - Allows for an overlay to assist with timing adjustment. Optimally adjust for your object to be in the middle of the overlay for most accurate results. Typically Vector delay is never needed.
    Color Delay (Index) float - Delays color pass.
    Vector Delay (Index) float - Delays vector pass.
    Reset: Page 1. Setup pulse - No definition provided

    2. Blur

    Complexity menu - Kernel chunk size of Optical Flow. 1x1 = Highest Quality 4x4 = Fastest
    Quality menu - Optical Flow's built in quality for Optimization.
    Gain float - Optical Flow's gain. Can be increased to make more extreme effects, but increasing Shutter Speed Length / Motion Blur Amount is recommended first.
    Optical Flow Threshold float - Helps filter out noise and artifacts from Optical Flow.
    Pre-Shrink float - Pre-Shrinks the resolution before Optical Flow TOP. (Very effective for optimization!)
    Shutter Speed Frame Rate menu - Current Perform FPS: Uses Perform FPS to help match dynamic FPS when rendering to Movie File Out TOPs. Project FPS: Matches Project Framerate Manual FPS: Set the frames per second manually
    Manual FPS int - Manual override of framerate for pseudo "FPS".
    Shutter Speed Length (s) float - Affects the length / intensity of the blur.

    Technically: 1 = 360 degree open shutter.
    Samples int - Affects the amount of pseudo frames that are samples and increases quality of the blur's complexity / smoothness.
    Gradient Magnitude (Blur Steepness) float - Controls the steepness of the falloff curve at the head and tail of motion blur streaks.
    Smooth Vector toggle - Controlled by macro. Enables blurring of vector map, which can decrease artifacts at the cost of details.
    Pre-Shrink int - Vector smoothing pre-shrink.
    Filter Size int - Vector smoothing filter-size.
    Refine Vector Edge toggle - Enables vector edge refinement which allows you to refine the smoothing from earlier. This is used to trim the blur back to reduce the blur from spilling onto the background / other GEOs.
    Initial Blur int - Overall initial blur.

    (Smoothens out the entire shape.)
    Threshold float - Threshold that trims back the initial blur.

    [Trims edges off of initial blur.]
    Soften float - Softens the edges of the threshold. If set too high, it will appear as YELLOW in the Vector Edge debug pass, which means that it is decreasing some of your blur. If set correctly, it will show as GREEN.

    [Smoothens out edges.]
    Post Blur int - Blurs the entire edge refinmeent pass to reduce sharp edges on vector map after the refinement blur>thresh process. Appears as RED in the Vector Edge debug pass.

    [Smoothens out final pass.]
    Reset: Page 2. Blur pulse - No definition provided

    3. Vectors

    Display Vector Debug toggle - Enables vector visualizer.
    Vector Color Mode menu - Changes Vector Debug color modes. Alpha will affect the blending between various modes.

    Constant: Single color [Vector Color] Input * Color: Multiplies [Vector Color] Vector Color: Displays RG Vector Map instead.
    Vector Color rgba4 - Changes color variable for Vector Debug modes.
    Vector Size float - Changes Vector Debug overlay sizes.
    Vector Style menu - Changes Vector Debug overlay styles.
    Vector Density float - Changes Vector Debug density (how many vector debugs are shown).
    Vector Max Line Length float - Changes Vector Debug maximum line length.
    Vector Background toggle - Enables background (useful for isolating Vector Debug for creative uses).
    Vector Background Color rgba4 - Background color for Vector Debug (alpha will control transparency).
    Reset: Page 3. Vectors pulse - No definition provided
